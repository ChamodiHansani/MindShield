🛡️ MindShield

Early Detection of Mental Health Risks from Sinhala + Singlish Social Media Posts
MindShield is an AI-driven solution designed to identify mental health risks in code-mixed (Sinhala and Singlish) social media content. By leveraging Deep Learning and Axiomatic Explainability, the system provides transparent risk assessments to assist community members and mental health professionals.

🚀 Key Features
Code-Mixed NLP: Optimized for Sinhala Unicode and phonetic Singlish (transliterated Sinhala).

Phonetic Normalization: Standardizes informal spelling variations to improve model consistency.

Axiomatic Explainability: Utilizes Integrated Gradients to highlight specific "risky words" that contribute to the prediction.

Risk Categorization: Classifies text into High Risk, Medium Risk, or No Risk.

🛠️ Tech Stack & Performance
Core Language: Python

Model Architecture: Hybrid Transformer-based Classifier

Model Accuracy: 88.4%

Explainability Technique: Integrated Gradients (Feature Attribution)

Environment: Developed and tested on Google Colab (NVIDIA T4 GPU)
